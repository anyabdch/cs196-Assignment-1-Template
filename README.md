# CS 196 Assignment-1 Template

This is the template for the first course assignment. From your repository, pull it to the IDE of your choice using the steps delineated by GitHub

Given that you all likely have differing experience levels with utilizing generative AI tools for coding, the first assignment will serve to give you baseline experience of what you can do with these tools and compare different model options. We will focus on test-driven development, test-last development, debugging, and natural language processing. As a helpful starting place, here are some slides Download some slideson the topics covered by the assignment. I encourage you to look through them, and jot down any questions they may bring up.

Your goal is not just to get working code, but to evaluate which models produce better, cleaner, and more accurate code. Some of these exercises will ask you to simply use Copilot (in or out of your IDE) or Cursor or another IDE-based tool to help you get comfortable with in-line code generation within your IDE. Some of these exercises will ask you to use LMArena (arena.ai). Instead of relying on a single AI tool, you will pit different AI models against each other. Log your observations in an Arena Log document and submit this with your finished code. 

## Get Set Up with LMArena:

- Make a copy of the Arena Log template
- Go to LMArena
- Hit the New Chat button
- You have two options for how you approach the Arena: 
-- Use the Battle mode. This mode gives you two anonymous answers side-by-side.
-- Use the Side by Side mode. This mode allows you to choose which models you want to compare. If you choose this option, please compare at least three different models.

Feel free to additionally continue exploring different models and tools beyond LMArena as you complete these exercises.
